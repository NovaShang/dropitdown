How it works

Drag → Markdown → your agent. Three steps.

1

Drop

Drag a file from anywhere — a chat download, a screenshot, a PDF — onto the DropItDown icon in your menu bar.

2

Convert & file

It's converted to clean Markdown on your Mac. An AI model (your key) files it into the right folder in your vault and writes a one-line summary.

3

Hand it to your AI

The note's path lands on your clipboard. Paste it into Claude Code or Cursor — your agent reads clean Markdown, not a binary PDF. Later, it can grep the vault to find anything.

Why it’s built this way

⚡

Convert once, locally

Conversion runs on your Mac — no API call, no upload. Your agent reads clean text instead of chewing a binary PDF, for a fraction of the tokens.

🤖

Agent-ready

One click writes a CLAUDE.md into your vault so any agent opened there knows how to search it. Just open Markdown — zero lock-in.

📝

Markdown by default

Every file becomes a paired .md note with YAML frontmatter (one-line summary + link back). Search it in any editor, or point notes at your Obsidian vault.

🧠

Learns from your fixes

Filed somewhere wrong? Tell it where it should have gone. The rule gets recorded so the same mistake doesn’t recur.

🏠

Files stay on your Mac

Originals and notes live in your own folders. Only a short text excerpt is sent for classification — to your provider, on your key.

🔌

Bring your own key

Classification runs on your own key — DeepSeek, OpenAI, Claude, or any OpenAI-compatible endpoint. No middleman, no account.

Browse what you’ve filed

Click the menu-bar icon for the main window — History and a Markdown vault browser.
